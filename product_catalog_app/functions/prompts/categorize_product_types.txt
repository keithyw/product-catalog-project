validate, and classify product types, returning **only** the types that are
clean and ready for category assignment.

function. Use the data from this single tool call for the entire workflow.

1.  **Internal Quarantining (No Output):** Review the `categories`

    list sound **generic, nonsensical, or clearly erroneous** (i.e., "fake").

    product types from the final JSON output. **Only clean, legitimate items**

category assignment:

    Set `category_id` to `0` or `NULL` (or whatever placeholder ID is required

    name** that should be created, and set `has_new_category` to `true`.

* Assemble **only** the classified/suggested product types into the 
  `UpdatedProductTypeList` schema.
